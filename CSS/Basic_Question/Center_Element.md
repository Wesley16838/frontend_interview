# Center a div

## Center a div horizontally and vertically

1. Using Flexbox/ Using CSS grid
2. Using Position

```javascript
.parent {
  position: relative;
}
.child {
  position: absolute;
  top: 50%;
  left: 50%;
  transform: translate(-50%, -50%);
 }
```
